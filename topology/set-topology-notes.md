# Topology Notes
拓扑本身就又是经验的，又是符号的。经验在于连续形变。

## 紧致Hausdorff定理
如果 $X$ 是紧致空间，$Y$ 是豪斯多夫 (Hausdorff) 空间，且 $f: X \to Y$ 是一个连续的双射 (Continuous bijection)，那么 $f$ 的逆映射必定连续，即 $f$ 必定是同胚映射。
反过来是不成立的。$X = [0, 1)$ 是 Hausdorff 空间（但不是紧致的，因为它缺少了一个端点）。我们的目标空间依然是圆圈 $Y = S^1$（它是紧致的）。
该怎么想象这个定理？$X$ 是紧致的（它天生偏弱，拓扑不能太强）。$Y$ 是 Hausdorff 的（它天生偏强，拓扑不能太弱）。连续性要求：天生偏弱的 $X \ge$ 天生偏强的 $Y$。所以他俩一样。
那这是否意味着同一个集合上，只能有一个又紧又Hausdoff的拓扑呢？并不是，因为两个拓扑有可能是不可比较的。如果有两个紧Haus的拓扑，其中一个完全包含另一个，那么他们俩就得是一样的。
